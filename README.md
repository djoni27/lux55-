# Platinum Store Pro - المتجر البلاتيني المطور

متجر إلكتروني متطور مع لوحة تحكم شاملة ومساعد ذكاء اصطناعي.

## المميزات

- **10 أنماط تصميم مختلفة**: من الكلاسيكي إلى العصري والنيون والزجاجي
- **مساعد ذكاء اصطناعي**: يساعد العملاء في اختيار المنتجات المناسبة باستخدام Gemini AI
- **دعم 3 لغات**: العربية، الإنجليزية، والفرنسية
- **قاعدة بيانات Supabase**: تخزين آمن ومتزامن للبيانات
- **لوحة تحكم متطورة**: إدارة كاملة للمنتجات والإعدادات
- **تصميم متجاوب**: يعمل بكفاءة على جميع الأجهزة
- **تكامل WhatsApp**: إرسال الطلبات مباشرة عبر واتساب
- **ضغط الصور التلقائي**: تحسين الأداء وسرعة التحميل

## المتطلبات

- Node.js (v16 أو أحدث)
- حساب Supabase (مُعد مسبقاً)
- مفتاح Gemini API

## التثبيت والتشغيل

1. تثبيت التبعيات:
```bash
npm install
```

2. تكوين متغيرات البيئة:
ملف `.env` محدث بالفعل بمعلومات Supabase و Gemini API

3. تشغيل المشروع:
```bash
npm run dev
```

4. بناء المشروع للنشر:
```bash
npm run build
```

## استخدام لوحة التحكم

1. انقر على زر الإعدادات في أسفل يمين الشاشة
2. أدخل كلمة المرور: `1234`
3. قم بتخصيص:
   - معلومات المتجر (الاسم، الشعار، رقم الواتساب)
   - التصميم والأنماط (10 أنماط مختلفة)
   - إضافة وتعديل المنتجات

## قاعدة البيانات

تم إنشاء الجداول التالية في Supabase:

- `store_settings`: إعدادات المتجر (الاسم، الشعار، الألوان، إلخ)
- `products`: المنتجات مع الصور والأسعار والفئات

جميع البيانات محمية بنظام Row Level Security (RLS).

## الملفات الرئيسية

- `index.html`: الواجهة الرئيسية مع 10 أنماط تصميم
- `index.tsx`: منطق التطبيق وتكامل Supabase و Google Gemini AI
- `styles.css`: الأنماط المخصصة والرسوم المتحركة
- `vite.config.ts`: إعدادات البناء

## مساعد الذكاء الاصطناعي

المساعد الذكي يستخدم Google Gemini AI مع وضع التفكير العميق لمساعدة العملاء في:
- اختيار المنتجات المناسبة
- الإجابة على الأسئلة حول المنتجات
- تقديم توصيات مخصصة

## الأنماط المتاحة

1. Minimal - تصميم بسيط ونظيف
2. 3D Pop - تأثيرات ثلاثية الأبعاد
3. Neon - أسلوب نيون مشرق
4. Glass - تأثير زجاجي شفاف
5. Luxury - تصميم فاخر بلون ذهبي
6. Comic - أسلوب الكوميكس
7. Gradient - تدرجات ملونة
8. Clay - تصميم طيني ناعم
9. Cyber - أسلوب سايبر بانك
10. Nature - تصميم طبيعي

## الدعم

للمساعدة أو الاستفسارات، راجع ملفات المشروع أو قم بفحص console المتصفح للأخطاء.
